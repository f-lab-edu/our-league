## 🔑주요 기능 소개

- 지역을 기반으로 서로 같은 운동 취미를 가진 사람을 연결해 준다.
- 대결 매칭을 진행하고 결과를 입력한다.
- 누적 결과를 바탕으로 지역 순위를 보여준다.
